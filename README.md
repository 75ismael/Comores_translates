# 🌍 Shikomori Linguistic Platform

Une plateforme intelligente dédiée à la préservation et à l'apprentissage de la langue **Shikomori** (Comores).

## 🚀 Fonctionnalités
- **Dictionnaire Intelligent** : Recherche bilingue Shikomori ↔️ Français.
- **API RESTful** : Backend robuste avec Django & Django REST Framework.
- **Interface Moderne** : Frontend ultra-rapide avec React et Tailwind CSS.
- **Panel d'Administration** : Gestion simplifiée des mots et validation des contributions.
- **Architecture Évolutive** : Prêt pour l'intégration de l'IA (Traduction NMT) et React Native.

## 🛠️ Stack Technique
- **Backend** : Python, Django, Django REST Framework.
- **Frontend** : React, TypeScript, Vite, Tailwind CSS.
- **Base de données** : SQLite (Dev).

## ⚙️ Installation & Lancement

### 1. Backend (Django)
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 manage.py migrate
python3 manage.py runserver
```

### 2. Frontend (React)
```bash
cd frontend-web
npm install
npm run dev
```

## 📈 Objectifs de Préservation
Ce projet vise à numériser le patrimoine linguistique comorien et à offrir des outils d'apprentissage accessibles à la diaspora et aux chercheurs.
